Для перемещения: cd Desktop/ 
Посмотрим содержимое папки Desktop с помощью команды ls (List Directories):

Для перехода на один уровень вверх, в родительский каталог, используется команда
cd ..
Путь к самый верхнему (корневому) каталогу файловой системы обозначается просто как "/":
Итак, команда cd позволяет переходить в каталоги несколькими способами:

Указывается путь от корневого каталога к целевой папке. Например: cd /etc/apache2/
Указывается путь от домашнего каталога пользователя. Например, cd ~/Desktop
Указывается путь от текущего каталога. Например, cd ./Desktop (данная команда эквивалентна команде cd Desktop или cd ./Desktop/../Desktop)

Вернуться в текущую cd~
Посмотреть путь pwd
Создать папку mkdir
Созать сразу все нужные папки  mkdir -p one two three
Несколько сразу вложенных папок Desktop $ mkdir -p {css/layout,html/pages,js/scripts/modals}

Очистим экран с помощью команды clear (эта команда пролистывает ранее введенные команды наверх за экран). Создадим новый пустой файл на рабочем столе с помощью команды touch:

Desktop $ touch test.txt

Также для создания файла и записи в него информации, которую выводит какая-либо команда, можно воспользоваться механизмом перенаправления вывода с помощью символа >. Например, следующая команда создаст на рабочем столе файл testLS.txt и запишет в него вывод команды ls -la:
Desktop $ ls -la > testLS.txt

Такой прием может быть полезен для дальнейшего анализа вывода определённой команды.

С помощью перенаправления вывода и команды echo без параметров можно создать пустой файл:

Desktop $ echo > echo.txt

Удалять файлы можно с помощью команды rm (Remove). Например, следующая команда удалит файл test.txt в текущем каталоге:

Desktop $ rm test.txt 
Можно удалять сразу несколько файлов:

Desktop $ rm testLS.txt echo.txt
Для удаления пустых папок используется команда rmdir (Remove Directory). Например, следующая команда удалить папку one из текущего каталога (при условии, что эта папка пуста, то есть не содержит файлов или вложенных папок):

Desktop $ rmdir one
Создадим новую папку new, перейдем в неё и создадим там файл index.html:

Desktop $ mkdir new
Desktop $ cd new
new $ touch index.html
Вернемся назад в папку Desktop и попытаемся удалить папку new:

new $ cd ..
Desktop $ rmdir new
rmdir: не удалось удалить 'new': каталог не пуст
Произошла ошибка, так как внутри папка new не пуста, в ней находится файл index.html.

Удалять не пустые папки можно с помощью команды rm с ключом -r (Recursive). Следующая команда удалит папку new вместе со всем ее содержимым:

Desktop $ rm -r new  


Давайте разберёмся, как можно в консоли копировать и перемещать файлы и папки.

Создадим папку test и файл index.html:

Desktop $ mkdir test
Desktop $ touch index.html
Переместим файл index.html в папку test. Для этого используется команда mv (Move):

Desktop $ mv index.html test 
Перейдем в папку test и с помощью той же команды mv переименуем файл index.html в another.html:

Desktop $ cd test 
test $ mv index.html another.html
Вернемся в папку Desktop и создадим в ней новый файл new.html:

test $ cd ..
Desktop $ touch new.html
Скопируем теперь файл new.html в папку test. Для этого используется команда cp (Copy Paste):

Desktop $ cp new.html test
Создадим ещё одну папку two и попробуем скопировать в нее папку test – копирование не выполнится:

Desktop $ mkdir two
Desktop $ cp test two 
cp: пропускается каталог 'test'

Для копирования папок со всем их содержимым нужно в команде cp указывать ключ -r (Recursive):

Desktop $ cp -r test two
Ключ -r можно использовать и в команде mv для перемещения файлов.

Рассмотрим еще один пример. Создадим новую папку project и три файла:

Desktop $ mkdir project
Desktop $ touch one.html two.html three.html
Переместим теперь все созданные файлы в папку project. Для этого можно воспользоваться шаблонным символом "*", замещающим любое количество любых символов, чтобы не писать в команде имена всех перемещаемых файлов.

Desktop $ mv *.html project
Шаблонный символ "*" можно применять не только в команде mv, но и в других командах для работы с файловой системой (cp, rm).

Рассмотрим команду find, с помощью которой можно выполнять поиск файлов по различным параметрам.
